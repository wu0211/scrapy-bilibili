# scrapy-selenium-proxyPools-bilibili

- 根据关联词列表爬取视频列表输出json文件


## 前提
1. 配置好搜索那些关键词  keyWords=[]
3. 配置好chromedriver



## 执行

2. `scrapy crawl b --nolog`
